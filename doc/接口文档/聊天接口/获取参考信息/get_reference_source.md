# 获取问题引用信息

获取问题的引用路径和页码信息

- Path： /v1/chat/{id}/reference
- Method：GET

- 请求路由参考示例：

|参数名      |描述 |
|----------- |----------- |
|id  |问题ID |

- 请求Query参数示例:

`如果是授权访问，无需添加Code查询参数，如果是匿名访问，需添加Code参数（在匿名发布应用的访问链接里）`

|参数名      |描述 |
|----------- |----------- |
|Code  |匿名应用Code |

- 响应参考示例：

    ```
    [
        {
         "source": "string",    // 参考路径, 存储key
         "page": 0,    // 页码
         "type": "image"    // 引用类型
        }
    ]
    ```
